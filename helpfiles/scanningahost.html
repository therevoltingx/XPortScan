Scanning a Host


To scan a host simply type in the address (i.e. yahoo.com) or ip (i.e. 127.0.0.1).
Then click on scan to start scanning.

There are a few options:

Scan Type:
You can specify which scan method to use.
(Note: for now only connect is supported)

Ports To Scan:
By default XportScan only scans the most used ports read from portlist.txt.
You can specify which port to scan by choosing custom. (Note: There are a maximum of 65535 ports in a system.)

Timeout
Not currently used.


Threads
Specify the maximum number of threads XportScan can use.


Raw Packet Delay
Not currently used.

Note:
XPortScan, by default, scans only ports 1-1024 plus the most common used ports read from the portlist.txt file.
You can upgrade the portlist.txt with any file that uses the same format (which is actually the nmap-services file)

Read the tips section for notes on the advanced options.
